<context>
La feature está desplegada y funcional. El flujo es:
1. Usuario abre imagen o carrusel
2. Ve botón "Generar idea para mí" (Wand2) en la zona de prompt
3. Al pulsar, llama a /api/generate-user-prompt con el brand kit activo
4. La API lee el sistema de prompts de Convex, inyecta variables del brand kit, llama al modelo configurado en admin
5. El texto generado aparece en el textarea

Los prompts son editables desde Admin → pestaña "Prompts".
El idioma de salida viene de brand_kit.preferred_language (configurable en /brand-kit).
</context>
